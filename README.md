# Dotfile manager
This repo contains my dotfiles as well as a simple cli tool to update local config with source files or reverse.

## Features of cli
- **Compare** - Compares local config vs source config using git diff
- **Update source** - Update the source config with new changes from local config
- **Multiple config** - Support different dotfiles types
- **Sync with source** - Update the local config with new values from source
- **Partial updates** - Review what would be updated and allow line by line approval
- **Nice TUI** - Nice looking TUI for operations
- **Configurable via env** - Control source and destination using environmentals
